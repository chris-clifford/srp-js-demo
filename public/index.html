SiRP Demo : Secure (interoperable) Remote Password Authentication

Usage

This is a demo of the SiRP Ruby Gem, and the JSRP Javascript client. The code for this example can be found on Github at grempe/sirp-demo

To use this demo, first register a username and password (don't use anything you would'nt want exposed please. This test service is not hardened.). Once that step is complete use the same credentials to login using SRP.

User Registration

In this step a new user is registered by generating a salt and verifier in the browser and posting those attributes to the /users endoint on the server.
